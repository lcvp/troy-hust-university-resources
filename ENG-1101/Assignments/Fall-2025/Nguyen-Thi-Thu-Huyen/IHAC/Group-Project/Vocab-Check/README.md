# Requirements

**Purpose:**  
Check classmates' memory of the words learned in previous reading lesson.

**Numbers of selected words:**  
15-20 new words each reading

**Format: choose one out of two:**  
1. Exercises in Word Files (like definition matching, gap filling, finding antonyms, etc.)  
2. Games on PowerPoint slides (like *Lucky number, Hidden pictures, Mystery Box, etc.*) or online apps (like Kahoot, Quiziz, Blooket, etc.)

**Content:**  
exercises/games checking:  
- Meaning  
- Spelling  
- Pronunciation  
- Usage of the words  

**Time:**  
Each group organizes the vocab check activity in around *15-20 minutes*

**Number of exercises/ games:**  
From 2-3
